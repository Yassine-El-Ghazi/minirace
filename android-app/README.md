# MineRace Wallet — Android

Kotlin/Jetpack Compose wallet that keeps the private key on-device (secp256k1,
stored encrypted via Android Keystore-backed EncryptedSharedPreferences).
Signs transactions locally and posts them to the MineRace server.

## Build

1. Open `android-app/` in Android Studio (Giraffe or newer).
2. Let it sync Gradle and download the SDK components.
3. `Run` → pick an emulator (API 26+) or a connected device.

From the command line (once Gradle wrapper is generated by Android Studio):

```bash
./gradlew assembleDebug      # builds the APK
./gradlew installDebug       # installs on connected device/emulator
```

## Server URL

- **Emulator:** defaults to `http://10.0.2.2:3000` (maps to host's `localhost`).
- **Physical device:** set it in Settings to `http://<your-laptop-LAN-IP>:3000`
  and make sure the laptop and phone are on the same WiFi.
